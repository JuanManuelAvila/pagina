27 That is a... / That is not a... Negation with the word not (Negación mediante la palabra not) This is a book. This is an eraser. This is not a book. This is not an eraser. Estas son oraciones afirmativas: indican que algo es verdad. Se logran añadiendo la palabra not después de is. That is a... / That is not a... This is a... / This is not a… Singular demonstrative pronouns: this and that (Pronombres demostrativos en singular: this y that) Las palabras this y that son pronombres demostrativos: muestran una relación espacial con un objeto. La palabra this indica que el objeto está cerca, mientras que la palabra that indica que está lejos. This is a book. That is a schoolbag. Is this a…? / Is this an…? / Is that a…? / Is that an…? Yes/no questions (Preguntas de sí o no) A: Is this a chair? B: Yes, it is . A: Is that a window? B: No, it’s not. That’s a picture. Estos son ejemplos de preguntas de sí o no. Se forman cambiando el orden del sujeto y la acción. Estas obtienen una respuesta afirmativa (sí) o negativa (no).
