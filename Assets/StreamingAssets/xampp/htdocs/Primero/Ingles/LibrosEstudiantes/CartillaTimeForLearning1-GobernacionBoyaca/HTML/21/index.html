Para saber más This is a... / This is an… Indefinite articles: a, an (artículos indefinidos: a, an) This is a book. This is a n eraser. This is a book. This is a picture. This is an eraser. This is an object. Las palabras a y an funcionan como artículos indefinidos, como un y una en español. El artículo indefinido a se utiliza cuando la siguiente palabra comienza por consonante. El artículo indefinido an se utiliza cuando la siguiente palabra comienza por vocal. Plurals Formation of plurals with -s ending (formación del plural mediante la terminación -s) La mayoría de los sustantivos (nombres de ideas, objetos, personas) tienen forma singular y plural . La forma singular hace referencia a un solo objeto. La forma plural hace referencia a más de un objeto. Generalmente, se debe añadir la terminación - s a la forma singular para formar el plural.
